Revamping UI/UX for productivity

PreForm is a Desktop Application that allows users to prepare 3D models for printing on Formlabs printers. The application is used by engineers, designers, and other professionals to create high-quality prints. The goal of the project was to create a new UI framework that would make the application more user-friendly and intuitive.

Objective: Create a modular framework and UI/UX elements, for a simpler and powerful user-experience to streamline workflows.

PreForm

Before

  • The application was not intuitive enough, and the user had to navigate through multiple screens to perform a single task.
  • In addition, visual clutter and lack of consistency in the UI elements made the application difficult to use.
  • Architecture was not modular and had a lot of copy-paste code.

After

  • The new UI framework is modular and allows for easy customization of UI elements.
  • It also has a consistent design language that makes the application easier to use.
  • Architecture was modularized and made it easier to maintain and extend.
  • Optimized QML code through profiling and refactoring, improving performance by 27%.

High Performance CAD Features

Led the end-to-end development of high-performance feature called part cages, a feature in PreForm, are used to efficiently pack and organize multiple small parts within a single 3D print. They create a structured enclosure around the parts, ensuring they remain in place during the printing process, particularly in SLS (Selective Laser Sintering) printing. This method minimizes the risk of part movement or damage, improves print reliability, and makes post-processing easier by keeping parts contained in a defined space.

Part Cages released in PreForm 3.41.0 , a high-performance feature that allows users to pack and cage, making it ease of post-process and eliminate pain points of 3D Printing.
