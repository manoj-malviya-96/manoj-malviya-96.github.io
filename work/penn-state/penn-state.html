Computational Design in 3D Printing

This research develops tools to automate design decisions for in-situ embedding in additive manufacturing, using two approaches—Shadow Projection and Voxel Simulation—to optimize orientation, design cavities, and shape converters, balancing design complexity and computational efficiency. Conference | Journal | Blog
Shows the design process of embedding an object during a 3D print. The object is placed in a cavity and covered with a shape converter, which provides a smooth surface for the 3D printer to print on.
